# IITBHU WiFi Auto Login

An Android app that automatically logs you into the **IITBHU Varanasi WiFi** and keeps your session active in the background.

## ✨ Features

- 🔐 Automatic login using your IIT(BHU) credentials
- 📶 Detects the captive portal automatically
- 🌐 Verifies internet connectivity after login
- 🔄 Keeps the session alive with automatic re-login
- 🚀 Starts automatically after reboot
- ⚡ Lightweight and battery-friendly
- 🔒 Credentials are securely stored on your device

## 📱 Installation

1. Download the latest APK.
2. Install the application.
3. Allow installation from unknown sources if prompted.
4. Enter your IITBHU LDAP credentials.
5. Enable Auto Login.

That's it! The app will automatically handle future WiFi logins.

---

## 🔄 Updating

Simply install the latest APK over the existing installation.
Your settings and credentials will be preserved.

---

## 🔒 Privacy

- Your credentials are stored **only on your device** using Android's encrypted storage.
- No passwords are shared with third parties.
- Anonymous diagnostics may be collected only to improve app reliability.
